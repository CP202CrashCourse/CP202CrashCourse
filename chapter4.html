Chapter 4t

Configuring a horizontal line

  • <hr>

CSS Border Property

  • Configures a border on the top, right, bottom, and sides of an element
  • Consists of; border-width, border-style, border-color
  • h2{border: 2px solid #ff0000}

Block/Inline Elements

  • Block display element has a default border width that extends to browsers margin
  • Inline display element has a border that closely outlines the element content
  • Browser display can often vary depending on which browser is being used
  • To configure sides on a border element you use border-bottom, border-left, border-right, or border-top
  • Another way to configure an underline is to assign a border-bottom

CSS Padding Property

  • Configures an empty space between the content of the HTML element and the border
  • Padding is set to 0px by default
  • Padding property can be configured using two values or percentages
    1. First value configures the top and bottom padding
    2. Second value configures left and right padding
  • Padding property can also be configured using four values or percentages
    1. Padding-top, padding-right, padding-bottom, padding-left

    Types of graphics commonly used on the web

    • GIF
      1. Graphics Interchange Format
      2. Best used for line art and logos
      3. Maximum of 256 colors
      4. Can be transparent
      5. Can be animated
      6. Uses lossless compression
      7. Can be interlaced
    • JPEG
      1. Joint Photographic Experts Group
      2. Best used for photographs
      3. Up to 16.7 colors
      4. Uses lossless compression
      5. Cannot be animated
      6. Cannot be made transparent
      7. Progressive JPEG – similar to interlaced display
    • PNG
    • Portable Network Graphic
    • Support millions of colors
    • Support multiple levels of transparency
    • Support interlacing
    • Uses lossless compression
    • Combines the best of GIF and JPEG
    • Browser support is growing

    HTML Image element

    • Configures graphics on a web page
    • Src attribute is the file name of graphic
    • Alt attribute configures an alternate text description
    • Height attribute configures the height of the graphic
    • Width attribute configures the width of the graphic
    • To create a image hyperlink, you use an anchor element to contain an image element

    Accessibility and Images

    • An alt description is necessary to convey meaning of graphic if the graphic isn't shown

    Thumbnail Image

    • A small image configured to link to a larger version of that image

    Image Optimization

    • The process of creating an image with the lowest possible file size that still renders a good quality image

    Image Editing Tools

    • GIMP
    • Adobe Fireworks
    • Adobe Photoshop

    When choosing names for image files
    • Use all lowercase letters
    • Do not use symbols or spaces
    • Do not change the file extension
    • Keep file names short and descriptive
    • Place images in their own folder
    • Code path to file in the src attribute

    A Progress element

    • Displays a bar that depicts a numerical value within a specified range

    CSS background-image property
    • Configures a background image
    • By default, background images repeat
    • Repeat-y – image repeats horizontally
    • Repeat-x – image repeats vertically

    CSS Sprite

    • An image that contains multiple small graphics that are configured as background images for various web page elements
