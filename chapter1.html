Chapter 1

The Internet

  • Interconnected network of computer networks that spans the globe
  • Began as a network to connect computers at research facilities so researchers at different places could communicate their findings
  • Works by connecting networks through numerous routes or paths so that when one is broken, it can be rerouted through another path
  • Developed by the Advanced Research Projects Agency
  • Four computers were connected by the end of 1969
  • In 1995 only 0.4% of the population was on the internet
  • There was a restriction on the commercial use of the internet until 1991 - this set the stage for ecommerce

World Wide Web

  • The graphical user interface to access info stored on computers running web servers connected to the internet
  • Tim Berners Lee created this to connect research between facilities
  • This used HTTP – hypertext transfer protocol to communicate between clients computers and web servers and HTML – hypertext markup language to format the documents

Mosaic – 1993

  • First graphical web browser – like chrome and firefox, etc

Internet Engineering Task Force (IETF)

  • Develop new internet protocol standard specifications
  • Open international community of designers, operators, vendors, and researchers concerned with the evolution of the operation of the internet
  • Security and routing

Internet Architecture Board (IAB)

  • Committee of the IETF which provides guidance and broad direction to the IETF
  • Create the new standards of the internet

Internet Corporation for Assigned Numbers and Names (ICANN)

  • Coordinate the assignment of internet domain names, IP numbers, protocol parameters, and protocol port numbers

Intranet

  • Private network that is contained within an organization or business
  • Shares info and resources among coworkers
  • There is a gateway to prevent unauthorized users from accessing

Extranet

  • Private network that securely shares part of the organizations info or operations with external partners like suppliers, vendors and customers
  • Virtual Private networks, digital certificates, and encryption messages are some privacies and securities used in extranets

W3C

  • Develops recommendations and prototype technologies related to the web
  • Web architecture, standards for web design, and accessibility
  • W3C Recommendations are created in working groups with input from many major corporations involved inbuilding web technologies
  • These recommendations are guidelines, not rules
  • Web Accessibility Initiative
  • Major area of work by the W3C
  • Develops recommendations for web content developers to facilitate use of the web by those with special needs (blind)
  • Ronald Mace created universal design

Questions to ask when determining whether the info on the web is reliable

  • Is the organization credible?
  • How recent is the information?
  • Are there links to additional resources?
  • Is it wikipedia?

World Intellectual Property Organization (WIPO)

  • Dedicated to protecting intellectual property rights internationally

Network

  • Two or more computers connected for the purpose of communicating or sharing resources
  • Networks include
  • Server computers
  • Receives requests from client computers for things such as resources
  • Client workstation computers
  • Computer work stations used by individuals such as personal computers on a desk
  • Shared devices such as printers
  • Networking devices and the media that connect them
  • Hubs and switches provide network connections for computers and routers direct info from one network to another
  • Media
  • Consists of copper cables, fiber optic cables, or wireless technologies

LAN – local area network

  • Confined to a single building or group of connected buildings
  • This is probably what laurier uses

WAN – wide area network

  • Geographically dispersed and uses some form of public or commercial communications network

Backbone

  • High capacity communication link that carries data gathered from smaller links that interconnect with it
  • Set of paths that local or regional networks connected to for long-distance interconnection

Client/server model

  • Client requests some type of service from the server and the server fulfills the request through a network
  • Multipurpose Internet Mail Extensions are rules that allow multimedia documents to be exchanged among many different computer systems
  • Auto, video, image, application, message, multipart, and text
  • Image/jpeg

Protocols

  • Rules that describe how clients and servers communicate with each other over a network
  • File transfer protocol
  • Allows files to be exchanged over the internet
  • Moves files from one computer to the next
  • Transfer web page files from computers to web servers
  • Email protocols
  • Simple mail transfer protocol, post office protocol, internet message access protocol
  • Transmission Control Protocol (TCP)
  • Official communication protocol of the internet
  • Ensures the integrity of the network communication
  • Breaks files and messages into individual units called packets
  • IP
  • Takes over after TCP using IP addresses to send each packet over the internet via the best route possible
  • IP address
  • Each device has a unique numeric IP address
  • Each consists of four groups of numbers called octets
  • Protocol version 4 uses 32 bit addressing where each group of number has a value between 0 to 255

Domain Name System

  • Associates IP addresses with the text-based URLs and domain names you type into a web browser address box

Uniform Resource Identifier

  • Identifies a resource on the internet

Uniform Resource Locator (URL)

  • Type of URI that represents the network location of a resource such as a webpage, graphic file, or mp3
  • Consists of a protocol, domain name, and location of file on web server

Domain Name

  • Locates an organization or other entity on the internet
  • Google.com
  • .com is the top level domain name
  • ICANN administers the top level domain name
  • Usually changes based on where one is using the internet (country)
  • Google is the second level domain name
  • www. Is the name of the web server
  • Subdomain is the separate website located at the same domain
  • Gmail.google.com for example
  • Fully qualified domain name
  • Combination of a host/subdomain, second level domain, and top level domain name

Markup Languages

  • Consist of sets of languages that tell the browser software how to display and manage a web document
  • Usually called tags and perform functions such as displaying graphics, formatting text, and referencing hyperlinks
  • SGML – standard generalized markup language
  • Not a document language but a description of how to specify on and create a document type definition
  • When Tim Berners lee created HTML, he used SGML to create the specification
  • HTML – hypertext markup language
  • Set of markup symbols or codes places in a file intended for display in a web browser
  • XML – extensible markup language
  • Developed by the W3C as a flexible method to create common information formats and share the format on the web
  • Not intended to replace HTML but to extend the power of it by separating the data from presentation
  • XHTML
  • Combines the formatting strengths of HTML and the data structure and extensibility strengths of XML

HTML5

  • Intended to be the successor of HTML4 and will replace XHTML
  • Supported by modern browsers but is not in final approved status
