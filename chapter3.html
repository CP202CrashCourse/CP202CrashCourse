Chapter 3

CSS – Cascading Style Sheets

  • A single cascading style sheet can control the color, typography, and layout of multiple web pages
  • Content is in HTML and styling is in CSS
  • Easier to look after site because it's DRY (Don't Repeat Yourself)
  • Types of Style Sheets – put the style for the body in the sections
  • Inline Styles (overwrites embedded styles because it is specified inline)(used for one element)
  • Colors are written in hexadecimal values where
  • Red = #FF0000
  • Green = #00FF00
  • Blue = #0000FF
  • Black = #000000
  • White = #FFFFFF
  • When choosing colors for text and background, you should use colors that have strong contrast to make the text easier to read – especially for color blind individuals
  • To make font a color
  • <p style="color:green">
  • Can also change the font appearance
  • Font-size
  • {p style="font-size:em3"}
  • Text Values Em Units Px Units Pt Units Percentage
    xx-small .5 em 8px; 6pt 50%
    x-small .6 em 11px 8pt 60%
    small .75 em 13px 10pt 75%
    medium 1 em 16px 12pt 100%
    large 1.15 em 18px 13.5pt 110%
    x-large 1.5 em 24px 18pt 150%
    xx-large 2 em 30px 24pt 200%

Font-family

  • <h1 Style="font-family:serif">
  • Changes the style of the font
  • Sometimes you will see multiple fonts written down - "font-family:serif, times new roman, veranda" - this is to make sure that if the program doesn’t have the first font, the second option will take over
  • Embedded Styles (styles the whole site - don’t have to keep styling each paragraph)
  • Put right into the HTML file
  • Creates style for all font
  • <style> P{color:blue; background:grey} </style>
  • There are also external styles and imported styles

In body{color:blue}

  • Body is the selector
  • Color is the declaration property
  • And blue is the declaration value

CSS Selectors

  • <h id="paragraph2"> This is a Heading</h>
  • <p class="bodyclass"> Paragraph </p>
  • In the <head> style section at the top of page, put #paragraph2 and give it a color
  • In the <head> style section at the top of the page, put p.bodyclass{color:turquoise}
  • This formats the color for the fonts for each paragraph in that particular section
  • You can also create a section and have only two p.bodyclass{color:turquoise} in the section and the other one outside of it
  • Section p.bodyclass{color:turquoise}
  • This makes only the paragraphs within the section turquoise while the other paragraphs remain the original color format
  • p.bodyclass span{color:pink} - in the head format at top of html page
  • <span> words you want pink within the other bodyclass turquoise</span>
  • To put a style sheet into a html page, you have to set up a CSS style sheet
  • <link rel="stylesheet" href="test1.css"/>
  • In purely CSS documents, you dont need and HTML codes
  • All you put is the styles – for example;
  • P{color;red}

To centre the page

  • Body{width:80%;margin-left: auto;margin-right: auto}
  • Don’t need margin-left or margin-right
  • {Margin: 0 auto}
  • 0 is the top and bottom, auto is left and right
