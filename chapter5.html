Chapter 5

Overall website design is related to the site purpose, you want to take into consideration the target audience

  • Hierarchical Design - starts with clearly designed home page with navigation links to major site sections
  • Often used for commercial and corporate websites
  • Within hierarchical website organization you want to make sure that there aren't too many main navigation links on the home page
  • You also want to make sure that there arent too many clicks needed to get to the needed page
  • Three click rule – a web page visitor should be able to get to any page on your website in a maximum of three clicks
  • Linear Design
  • Usually used for sequential viewing
  • Series of pages that provide a tutorial, tour, and presentation
  • Random Design
  • Sometimes called web organization
  • There is no clear path through the site
  • Might be used with artistic or concept sites
  • Not usually used for commercial sites

Design Principles

  • Repetition
  • Repeat visual elements throughout design
  • Contrast
  • Add visual excitement and draw attention
  • Proximity
  • Group related items
  • Alignment
  • Align elements to create visual unity
  • Provide Accessibility
  • Four people who benefit from accessibility
    1. A person with a physical disability
    2. A person with a slow internet connection
    3. A person using an old, out-dated computer
    4. A person using a mobile phone
  • Web Content Accessibility Guidelines 2.0
  • POUR
    1. Perceivable – content must be perceivable
    2. Operable – interface components must be operable
    3. Understandable – content and controls must be understandable
    4. Robust – content should be robust enough to work with current and future user agents including assistive technologies
  • When writing for the web
    1. Avoid long blocks of text
    2. Use bullet points
    3. Use headings and subheadings
    4. Use short paragraphs
    5. Check your spelling
    6. Use easy to read text fonts
    7. Use common fonts - Arial, Helvetica, Verdana, Times New Roman
    8. Use appropriate text size - medium, 1em, 100%
    9. Use strong Contrast between text and background
    10. Use columns instead of wide areas of horizontal text
    11. Carefully choose text in hyperlinks
    12. Avoid "click here" and make sure your hyperlinks are key words or phrases and not entire sentences

Color Theory

  • The study of color and its use in design
  • Color wheel
  • Primary colors
  • Secondary colors
  • Tertiary colors
  • There are color schemes based on the color wheel
  • Monochromatic – shades, tints, or tones of the same color
  • Analogous – a main color and two colors adjacent to it on the color wheel
  • Complementary – two colors that are opposite to each other on the color wheel
  • Split complementary – a main color, the color opposite it and two colors adjacent to the opposite color
  • Triadic – three colors that are equidistant on the color wheel
  • Tetradic – two complementary color pairs
  • Choose one color to be dominant, then use colors in the color scheme as accent colors
  • Use the accent colors for headings, subheadings, borders, and list markers etc
  • Use neutrals such as white, off white, grey, black or brown
  • Do not restrict yourself to web safe colors
  • Use tints, shades, or tones of colors

Graphic Design best practices

  • Be careful with large graphics – there is a 60k recommendation
  • Use the alt attribute to supply descriptive alternate text
  • Make sure that the message gets across even when the image is not displayed – have plain text to describe the image
  • Only use animation if it makes the webpage more effective
  • Use only necessary images, reuse images, and size images as small as possible and with acceptable display quality
  • Take load time into account
  • Try to limit the web page document to under 60k on the home page
  • Test screen resolution with multiple web browsers
  • Test at various screen resolutions
  • Design to look good at various screen resolutions – centered page content, set to either a fixed or percentage width

Wireframe

  • Sketch or blueprint of a webpage
  • Shows the structure of the page elements
  • Header, navigation, content, footer, and image locations
  • Place the most important info above the fold
  • Use adequate white or blank space
  • Use an interesting page layout
