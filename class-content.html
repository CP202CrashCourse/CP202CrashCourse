Class Content

Here you will find a collection of class notes, textbook reviews, and other work notes from students of CP202. Each submission is sorted through to find new and valuable content. When new information is gathered, we make additions to the master notes which you will find in each chapter to the left. We recommend that you start with Chapter 1 to gain a basic understanding of website design before you dive into the other chapters. As you go through the chapter notes, you will notice that some chapters reference others and there will be links leading you to aiding information. It is completely under your discretion whether or not you follow these links or continue on with the chapter. However, if you are having trouble understanding the content we strongly recommend you follow the aiding links. At the end of each chapter, you will have the opportunity to solidify information through learning activities. You will also be given the option of taking practice tests at the conclusion of each chapter. To better prepare yourself for class exams, we strongly recommend you end each chapter with the learning activities and practice tests provided.

Chapter 1

This chapter focuses on the internet, how it began, why it began, and the basic processes involved with it's functioning.

Chapter 2

This chapter introduces you to HTML coding, including defining tags with a doc type statement, creating headings and paragraphs in your document's body, and creating lists.

Chapter 3

This chapter introduces you to CSS style sheets and CSS selectors, as well as how to reference a style sheet in an HTML document.

Chapter 4

This chapter focuses on formatting lines, borders, and other image elements that are useful when separating your web page into distinct section.

Chapter 5

This chapter focuses on common types of website design, the basic principles of visual design, and best practices of universal web design.

Chapter 6

This chapter introduces you to the CSS box model, how to configure margins, as well as how to configure navigation in unordered lists.

Chapter 7

This chapter introduces you to best mobile design practices, how to configure a three-column page layout, and how to configure CSS for printing a web page.

Chapter 8

This chapter introduces you to creating basic tables within CSS, including table sections and increasing accessibility of tables.
