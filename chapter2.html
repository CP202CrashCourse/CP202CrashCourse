Chapter 2

To make a comment in coding

  • <!-- this is a comment --> * this does not show up in the website, only in the coding

To make an anchor(reference to other material)

  • <a href="http://website.com"> Word you want to direct you to content</a>
  • Absolute link – link to another website (has http or https in the link)
  • Relative link – ljnk to your own site (does not have http or https in the link)
  • Email hyperlink
  • <a href="mailto:me@gmail.ca"> me@gmail.ca </a>
  • People don’t do this as much because it’s an easy way for hackers to spam your email

To make a paragraph

  • <p> content </p>

To input an image

  • <img src="link to picture" /> </li>

HTML – set of markup symbols or code places in a file intended for display on the web

  • Thing that drives the web
  • Content is written in html
  • W3c sets the standard for html
  • HTML elements
  • Each markup is an HTML element and the things within the markup are the content
  • Each element has a purpose and each element has a tag that is enclosed with a less than or greater symbols
  • The closing tag always has a slash in front of the element
  • HTML 5 is the newest version that is supported by modern browsers

Document Type Definition

  • Formally define tags with a doc type statement
  • This identifies the type of markup being used <!DOCTYPE html>
  • <head> - contains info that describes the document (the description of the website found on the tab at the top of browser)
  • <body> - contains info for the content of the page
  • There are multiple headings that can be added to a document
  • <h1> Frogs </h1>
  • <h2> types of frogs </h2>
  • Paragraph elements allow you to create a new paragraph and separate content on the page
  • <p> new paragraph </p> - this sections things together
  • Line breaks just continues the text on the next line of the page
  • There is a stand alone tag for the breaks, only one tag
  • Text 1<br />
  • Text 2
  • Block quotes indent a quote for emphasis on the quote
  • <blockquote> text goes here </blockquote>
  • Other important tags for text
  • <b> – bold text
  • <em> – emphasized text
  • <i> – italicized text
  • <mark> – highlighted text
  • <small> – small fine print text
  • <strong> – stand out bold text
  • <sub> – displays a subscript
  • <sup> – displays a superscript
  • To underling something
  • <u> word to be underlined </u>
  • You can nest tags inside one another
  • <p><i>This text is italicized<strong>this text is bolded, and italicized</strong></i></p>

Lists

  • Unordered list – bullet list
  • <ulgt - contains the unordered list </li>
  • <li> - contains an item on the list </li>
  • <ul> </li>
  • <li> first item </li> </li>
  • <li> second item </li>
  • </ul>
  • Ordered list – numbered list
  • <ol> contains the ordered list
  • Descriptive list

Special characters have specific codes as well

  • © - & copy;
  • < - & lt;
  • > - & gt;
  • & - & amp;
  • & nbsp; *this is the code to put a guaranteed space between text on the page

Div Element – are only used for format

  • Creates a structural block area or division on a webpage with empty space above and below
  • <div> content </div>

HTML Structural Elements

  • <header> </header> – contains the webpage documents headings
  • <nav> </nav> - contains the webpage documents main navigation
  • <main> </main> – contains the webpage documents main content
  • <footer> </footer> – contains the webpage documents footer

Writing valid codes

  • http://validator.w3.org
  • Before handing something in – validate it
